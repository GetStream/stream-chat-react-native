Component to render custom attachments within the [`MessageList`](./message-list.mdx).
Please check the guides for [Custom Attachment](../guides/custom-attachment.mdx) for basic usage.

## Props

### <div class="label description">_overrides the value from [MessagesContext](../../contexts/messages-context#additionaltouchableprops)_</div> `additionalTouchableProps` {#additionaltouchableprops}

<AdditionalTouchableProps />

### `author_name`

Author name, to be rendered in footer of card.

| Type   |
| ------ |
| string |

### `image_url`

URL of image to render as cover image for card. In absence of `image_url`, `thumb_url` will be used for cover.

| Type   |
| ------ |
| string |

### `og_scrape_url`

URL to redirect to, when user presses on card.

| Type   |
| ------ |
| string |

<OnPressIn />

### `text`

Text description for card, which will be rendered in footer of card.

| Type   |
| ------ |
| string |

### `thumb_url`

Thumbnail URL for cover image.

| Type   |
| ------ |
| string |

### `title`

Title for card, which will be rendered in footer of card

| Type   |
| ------ |
| string |
