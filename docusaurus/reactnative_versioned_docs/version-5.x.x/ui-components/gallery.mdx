Component to render image attachments within the [`MessageList`](./message-list.mdx).

## Basic Usage

You can customize the `Gallery` component and provide it back to the SDK via the [`Gallery`](../../core-components/channel#gallery) prop on `Channel` if desired.

```tsx
import { Channel, GalleryÂ } from 'stream-chat-react-native';

<Channel Gallery={() => <Gallery preventPress />}
```
