**Note:** This prop is available only in SDK version >= [v3.9.0](https://github.com/GetStream/stream-chat-react-native/releases)

Throttling interval (in milliseconds) for react state update for channel, when new messages arrive in chat. By default state updates are throttled at 500ms, but for high concurrency
application this prop should be adjusted to 1000ms to avoid exccesive overload on JS thread.

| Type   | Default |
| ------ | ------- |
| number | 500     |
