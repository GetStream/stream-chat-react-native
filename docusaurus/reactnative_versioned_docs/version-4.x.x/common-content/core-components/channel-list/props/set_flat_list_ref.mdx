Callback function to access the underlying [FlatList](https://reactnative.dev/docs/flatlist) ref.

#### Example

```tsx
const flatListRef = useRef();

<ChannelList setFlatListRef={ref => (flatListRef.current = ref)} />;
```

| Type     |
| -------- |
| function |

| Parameter | Description  |
| --------- | ------------ |
| ref       | FlatList ref |
