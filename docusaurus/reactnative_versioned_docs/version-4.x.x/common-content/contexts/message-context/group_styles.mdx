Position of message in the group. A message group is a group of consecutive messages from the same user. _groupStyles_ can be used to style the message as per their position in message group.

e.g. user avatar (to which message belongs to) is only showed for last (bottom) message in group.

| Type                                               |
| -------------------------------------------------- |
| array of enum('top', 'bottom', 'middle', 'single') |
