The `ChannelPreviewMessage` is the UI component for channel previews messages. Renders the message preview for each channel on a list.

![](../assets/api-references/components/channel-list/visual_guide_1.png) <!-- Replace with propert screenshot -->

## UI Customizations

You can have a custom channel preview message component by setting the `ChannelPreviewMessage` prop on any of its parent UI components.

```jsx
import { ChannelPreviewMessenger } from 'stream-chat-react-native';

const CustomPreviewMessage = ({ lastMessagePreview }) => (
  <Text>text: {lastMessagePreview.previews.text}</Text>
)

<ChannelPreviewMessenger PreviewMessage={CustomPreviewMessage} />
```
