`SendButton` is the default UI component for "send message" button within `MessageInput`. Please check the guide of [Customize Message Input](../guides/customize-message-input) for usage of this component.

You can replace this default UI component using the `SendButton` prop on [Channel](../core-components/channel) component.

```tsx
<Channel SendButton={CustomSendButton} />
```
