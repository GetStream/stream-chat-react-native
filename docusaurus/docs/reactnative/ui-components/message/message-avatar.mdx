Component to render the avatar of the sender (user) of a message within the [`MessageList`](../../message-list).

## Basic Usage

You can customize the `MessageAvatar` component and provide it back to the SDK via the [`MessageAvatar`](../../../ui-components/channel#messageavatar) prop on `Channel` if desired.

Reducing the size of user avatar can be easily accomplished.

```tsx
import {Channel, MessageAvatar} from 'stream-chat-react-native';

const SmallerMessageAvatar = () => <MessageAvatar size={30} />

<Channel MessageAvatar={SmallerMessageAvatar} />
```

### `size`

Dimension for avatar image.

:::tip
You can also set the size for message avatar, using our [theming system](../../theme).

```tsx
const theme = {
  avatar: {
    BASE_AVATAR_SIZE: 30,
  },
};
```

:::

| Type   | Default |
| ------ | ------- |
| Number | 32      |
