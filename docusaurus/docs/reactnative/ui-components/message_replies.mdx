A component to render thread reply count and avatars of users who thread-replied to a message within the [`MessageList`](./message_list.mdx).
This is the default component provided to the prop [`MessageReplies`](../core-components/channel.mdx#messagereplies) on the `Channel` component.

## Props

### <div class="label required">required</div> **noBorder**

If true, little curved line next to the reply count won't be rendered.

| Type |
| - |
| boolean |

### <div class="label required">required</div> **repliesCurveColor**

Color for the curved line next to the reply count.

| Type |
| - |
| string |
