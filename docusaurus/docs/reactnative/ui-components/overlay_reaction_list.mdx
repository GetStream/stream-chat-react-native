UI component to render the reaction selector within message overlay.

You can replace this default UI component using the `OverlayReactionList` prop on [Channel](../../core-components/channel) component.

```tsx
<Channel OverlayReactionList={CustomOverlayReactionList} />
```

### data

_(Overrides the value from [MessageOverlayContext](../../contexts/message-overlay-context#data))_
