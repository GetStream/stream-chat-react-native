`MessageContent` is a child component of `MessageSimple`, and renders all the actual content of the message such as text, attachments, thread replies count etc.

:::note
Avatar of sender of message and reactions are not part of `MessageContent`
:::

## Basic Usage

**Use Case**: Turn off `onLongPress` handler for `MessageContent`.

```tsx
import {Channel, MessageContent} from 'stream-chat-react-native';

const CustomMessageContent = () => <MessageContent onLongPress={() => null} />

<Channel MessageContent={CustomMessageContent} />
```
