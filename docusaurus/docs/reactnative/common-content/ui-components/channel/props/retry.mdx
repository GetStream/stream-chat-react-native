Override of the _Resend Message_ [`MessageAction`](../../../../state-and-offline-support/object-types.mdx#messageaction) shown when a user long presses a message.
The `retry` function receives as a parameter the relevant message and must return a `MessageAction`.
Please refer to [the guide on customizing message actions](../../../../ui-cookbook/custom-message-actions.mdx) for details.

| Type             |
| ---------------- |
| function \| null |

| Parameter | Description                     |
| --------- | ------------------------------- |
| message   | message the action is called on |
