Removes an attached image in a message being composed in the `MessageInput` component. By default, this function is attached to `onPress` handler for close button in [ImageUploadPreview](../../../ui-components/channel/channel.mdx#imageuploadpreview).
It takes a string ID of the image to be removed in the [`imageUploads`](../../../ui-components/contexts/message-input-context.mdx#imageuploads) array as parameter.
