Removes attached file in message, which is being composed in `MessageInput` component. By default, this function is attached to `onPress` handler for close button in [FileUploadPreview](../../../ui-components/channel/channel.mdx#fileuploadpreview).
It takes a string ID of the file to be removed in the [`fileUploads`](../../../ui-components/contexts/message-input-context.mdx#fileuploads) array as parameter.
